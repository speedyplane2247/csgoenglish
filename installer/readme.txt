This is the installer, to automatically install the csgo_english.txt files.

(As well as an uninstaller)


There will be a few versions:

Batch (Windows): The easiest installer (and simplest)
Batch (Windows, EXE form): An easier to use version
Shell Script (OSX): Used for Mac
Shell Script (Linux): Used for Linux

A Windows GUI for changing between csgo_*.txt files.
